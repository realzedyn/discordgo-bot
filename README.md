# discordgo-bot
Yarıda bırakılmış bir Discord sunucu projesi kapsamında geliştirdiğim özel Discord botunu açık kaynak olarak paylaşmak istedim. Proje geliştirme sürecindeyken rafa kaldırıldığı için tamamlanamadı, ancak ortaya çıkan kodların Go (discordgo) ile bot geliştiren kişiler için faydalı olabileceğini düşünüyorum.

Bot, ileride çok yoğun işlemler gerçekleştirmesi planlandığından performans odaklı olarak geliştirildi. Bu ağır sistemler projeye eklenemeden geliştirme süreci sona erdi. Yine de mevcut haliyle proje; mimari yapı, performans optimizasyonları ve bazı geliştirme yaklaşımları açısından referans niteliği taşıyabilir.

Kodları inceleyerek mevcut özellikleri görebilir, ihtiyaçlarınıza göre düzenleyebilir ve dilediğiniz şekilde kullanabilirsiniz.
